# Sampled-coins-Pictures


Hier speichere ich Fotos, damit ich sie in meiner Excel-Tabelle verlinken kann.

## 🚀 Anleitung für den Link
1. Klicke oben auf ein Bild (z.B. bild1.jpg).
2. Klicke auf den Knopf **"Raw"**.
3. Kopiere die Internetadresse oben im Browser.
4. Füge sie in Excel ein mit der Formel: `=BILD("HIER_LINK_EINFÜGEN")`
